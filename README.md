#### Технологии используемые в проекте: 
* HTML;
* CSS;
* Grid-layout;
* Flexbox;
* Методология БЭМ, файловая структура Nested;

#### Требования к проекту

Проект должен быть резиновым, адаптивным, и не ломаться между точек перелома.
